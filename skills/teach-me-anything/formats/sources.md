# Sources format

**Sources** are the trusted material teaching is drawn from. The rule is *teach from sources, not memory*: before teaching a topic, find high-trust material for it and record it here, so lessons can cite it.

This is the Markdown that goes inside the `<script type="text/markdown">` block of `sources.html`.

## Shape

```markdown
# Sources: <topic>

## Primary
<The best few things overall — the ones you'd stake the teaching on. For each:>

- **<title>** — <author / who made it> · <type: book / paper / video / docs / course>
  <link>
  _Trust:_ <why this is credible — author's standing, peer review, official docs, reputation>
  _Use for:_ <which lessons or topics this backs>

## Supporting
<Solid material for specific corners of the topic. Same fields as above.>

## Practice communities
<Places the learner can test skills against real people — forums, subreddits,
classes, local groups. For each: name, link, and why it's well-regarded.>

## Rejected / low-trust
<Things you looked at and chose not to rely on, with a one-line reason. Keeps you
from re-evaluating the same weak source twice.>
```

## Rules

- Rank by **trust**, not convenience. Prefer primary sources, official docs, and named experts over aggregators and anonymous posts.
- Every lesson names one **primary source**. It should come from this list.
- Note *why* each source is trustworthy — that judgement is the point of the file.
- When a source turns out to be wrong or dated, move it to **Rejected** with a reason rather than deleting it.
